About Chat Application

This is a real-time chat application built using Django and Django Channels. The application supports both private and group chats with features such as message tracking, online status, and real-time updates.

Features

  • Real-time private and group chat functionality using WebSockets.
  • User authentication and profile management.
  • Creation and management of chat groups with admin access.
  • Tracking of online status and last seen times for users.
  • Management of group join requests with real-time updates.
  • Dynamic updating of message read status and seen counts.

Components

The project consists of the following main components:

  • Models: Define the database structure for users, groups, messages, and user activities.
  • Views: Handle the HTTP requests and render the appropriate templates.
  • Templates: HTML files that define the structure and layout of the web pages.
  • Consumers: WebSocket consumers that manage real-time communication for chat functionality.
  • Signals: Django signals that handle asynchronous updates and notifications.

How It Works

Here's a brief overview of how the application works:

  • Users can sign up, log in, and manage their profiles.
  • Authenticated users can create and join chat groups.
  • Users can send and receive messages in real-time, with messages being displayed instantly in the chat interface.
  • The online status and last seen time of users are tracked and updated dynamically.
  • Group join requests can be sent and managed in real-time, with users receiving instant notifications of request status changes.
  • Message read statuses are tracked, and users can see who has read their messages.

Technologies Used

  • Django: For building the backend and managing database interactions.
  • Django Channels: For handling WebSocket connections and real-time communication.
  • Bootstrap: For responsive and modern UI design.
  • JavaScript: For client-side scripting and dynamic updates.
